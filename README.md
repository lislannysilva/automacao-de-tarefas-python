# 🤖 Automação de Cadastro de Produtos com Python

Script de automação desenvolvido em **Python** para cadastrar produtos automaticamente em um sistema web.

O projeto utiliza **PyAutoGUI** para controlar o mouse e teclado e **Pandas** para ler uma base de dados em CSV com as informações dos produtos.

## 🚀 Funcionalidades

* Abrir o navegador
* Acessar o sistema e realizar login
* Ler a base de dados (`produtos.csv`)
* Preencher automaticamente o formulário
* Cadastrar todos os produtos da base

## 🛠 Tecnologias

* Python
* PyAutoGUI
* Pandas

## ▶️ Como executar

Instale as dependências:

```
pip install pyautogui pandas
```

Execute o script:

```
python codigo.py
```

⚠️ Durante a execução não utilize o mouse ou teclado.
